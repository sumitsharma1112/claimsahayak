# ClaimSahayak Official Rule Book — Knowledge Base

**Version:** ✅ **1.0 — RELEASED 10.07.2026** (Topics 1–12 approved & locked). Master compilation: `official-rule-book/RULE-BOOK-v1.0.md`.
**Scope:** India Post Savings Bank (POSB) / National (Small) Savings Schemes — **Deceased Claim Settlement only**
**Purpose:** Single source of truth from which the ClaimSahayak Rule Engine, Wizard, checklists, forms and PDF outputs will later be generated. Every rule in the Official Rule Book is traceable to an official India Post / Government of India publication.

---

## Golden Rule

> Official India Post / Government of India publications ALWAYS override every other source.
> No rule enters the Official Rule Book unless verified from an official publication.
> If verification cannot be completed, the rule is marked **UNVERIFIED** and excluded from the Official Rule Book.

## Source hierarchy (highest first)

1. **Government Savings Promotion Act, 1873** (as amended by Finance Act, 2018) — statute
2. **Government Savings Promotion General Rules, 2018 (GSPR)** + Gazette amendment notifications (G.S.R.) — statutory rules
3. **Scheme Rules 2019** (POSA, NS-RD, NS-TD, NS-MIS, SCSS, PPF, NSC, KVP, SSA schemes) — statutory schemes
4. **SB Orders / Directorate circulars** (Department of Posts, F.S. Division)
5. **POSB (CBS) Manual / POSB Manual Vol I & II** (as amended by SB Orders)

Reference sources (PostalStudy, PO Tools, SAPost, StaffNews, GovtStaff, blogs, YouTube, forums) are used **only for discovery** of official documents — never as the source of a rule.

## Rule record conventions

- Rule IDs: `CS-NOM-###` (Nomination), later `CS-MNM-`, `CS-MIN-`, `CS-NON-`, `CS-LEV-`, `CS-FIN-` etc.
- `Verification Status`: `VERIFIED` (official text in hand) / `VERIFIED-SUPERSEDED` (official but overtaken) / `UNVERIFIED` (reference-source only — never part of the Official Rule Book).
- Superseded rules are retained with pointers, never deleted (see `superseded-orders/`).
