# Rule 60, POSB (CBS) Manual / Rule 87, POSB Manual Vol I

**Title:** Payment of the Amount of National (Small) Savings Schemes Accounts/Certificates in the Name of Deceased Depositor(s)
**Source of current text:** Enclosure to **SB Order 31/2020 dt. 28.08.2020** (which substituted the rule text with immediate effect and **deleted** Rule 165 of POSB(CBS) Manual and Rule 50 of POSB Manual Vol II to avoid duplication). Sanction-limit table in sub-rule (4)(B) **revised by SB Order 36/2020 dt. 06.11.2020**. Operational clarifications: Addendum dt. 16.09.2020 (witnesses), SB Order 01/2023 (timelines/no sanction memo/no PRI-SDI verification).
**Archived:** `sources/sb-order-31-2020.pdf` (full 20-page text incl. forms) — full verbatim text preserved there; structured digest below.
**Caveat:** The POSB (CBS) Manual "corrected up to 31.12.2021" (circulated by SB Order 03/2022) integrates this text; official copy of the consolidated manual is still pending (`missing-documents/`). Until obtained, the SB-Order chain above is the authoritative text.

## Structure of Rule 60 (as revised)

| Sub-rule | Subject |
|---|---|
| (1) | Basis of settlement: (a) nomination, (b) legal evidence, (c) without legal evidence at discretion up to **₹5,00,000** (MoF limit). Limit applies **per account / per certificate registration number**. Minor/unsound-mind depositor dies without nomination → payment to guardian on claim + death certificate |
| (2) | **Nomination claims** — entitlement immediate, any amount, no legal evidence; procedure (i)–(xii) + Notes 1–5 (see Topic 1 rule CS-NOM-019/020) |
| (3) | **Legal-evidence claims** (succession certificate / probate / LoA): sanction powers — TS/LSG/HSG-II SPMs up to their 4(B) limits, beyond → **Divisional Head (no limit)**; HSG-I/MDG/HO/GPO postmasters — **no limit**; verification checklist; certified/self-attested copies with original comparison |
| (4)(A) | **No nomination & no legal evidence, ≤ ₹5,00,000:** claim after **six months** from death with death certificate, original passbook/certificate, **Affidavit Form-13**, **Disclaimer Form-14**, **Indemnity Bond Form-15** — Forms 13/14/15 on **non-judicial stamp paper** per the State's Stamp Act; claimant must be the person entitled under personal law (Hindu Succession Act ss.8/15, Mohammedan law, Indian Succession Act 1925); guardianship certificate for minor relatives if claimant is not natural guardian |
| (4)(B) | Sanction-limit table — see `financial-powers/` (as revised by 36/2020) |
| (4)(C) | Special-feature cases / lacunae → refer to **Directorate** |
| (4)(D)-(E) | Scrutiny duties of sanctioning authority; powers beyond ₹10,000 exercised personally; defect handling; **no further verification** when witnesses accepted + ID/address proof + death proof + attested annexures present |
| (5) | Acceptable death-certificate issuers (municipality/hospital/police; Gazetted Officer/MP/MLA/Panchayat officer/Mukhiya where unavailable; employer/doctor ≤ ₹500; Parsee panchayat/church burial) |
| (6) | Guidelines to sanctioning authorities: Act ss.4A & 5 + GSPR 15(6)(i) give **full discretion + immunity** for bona fide payments; satisfaction from **two witnesses + claimant ID/address proof**, no further enquiry; natural guardians of minor claimants need **no guardianship certificate**; missing passbook → Divisional Head orders closure per Rule 37(3) CBS Manual |
| (7) | Payment to minor claimants; guardian = father/mother, else person entitled to care of property; "minor alive" certificate; mother of Muslim minor valid guardian (Act s.3(h)); payment directly to minor is **not** a valid discharge |
| (8) | Presumed death — holder not heard of for **7 years**; claimant indemnifies Government |
| (9) | Claimants/relatives abroad — consular authentication / Notaries Act 1952 reciprocity (UK, Hungary, Ireland, Belgium, New Zealand) / **Hague Apostille** country list; payment only to POA holder in India |
| (10) | **Interest & continuation table** (scheme-wise gist — see below); claimant presents sanction + passbook; payment **only by crossed cheque or credit to savings/bank account** |
| (11) | **Transfer of account/certificate to claimant's name** (continuation till maturity): new AOF (+ specimen signatures for BO accounts); CIF created/changed in CBS |
| (12) | Same-day **acknowledgment** of claim request (format prescribed) |
| (13) | **Register of deceased claim cases** (12-column format); disposal within **7 working days**; monthly review; inspection checks |
| (14) | Branch Office procedure: GDS BPM accepts claim + witnesses, notes death against DLT & specimen signature book, blocks transactions, submits to Account Office |
| (15) | **Reconciliation certificate** for name mismatch — issued by Divisional Head/Head of GPO/Gazetted HO on declaration + evidence + two witnesses |

## Sub-rule (10) — scheme-wise options for nominee/legal heir (gist table verbatim substance)

| Scheme | Options for nominee/claimant | Interest on deceased claim |
|---|---|---|
| SB (POSA) | Close the account | Up to preceding month of closure |
| RD | Close and take payment immediately, **or** transfer account to own name and continue till maturity | Full maturity value if continued; else per Rule 12 table, NS-RD Rules 2019 |
| TD | Close immediately **or** continue till maturity in own name | TD rate for completed years (within TD tenor); POSA rate for completed months beyond |
| MIS | Close any time | Interest up to month preceding refund |
| SCSS | Spouse **sole nominee** meeting eligibility may transfer account to own name and continue (if no second SCSS account); else close | SCSS rate up to date of death; thereafter POSA rate until payment |
| PPF | Close any time | PPF rate up to end of month preceding payment |
| SSA | Close any time | SSA rate up to date of death; thereafter POSA rate |
| KVP | Close immediately **or** continue till maturity in own name | Full maturity value if continued; POSA rate if closed < 2½ yrs; KVP Rules 2019 Rule 6 table thereafter |
| NSC | Close immediately **or** continue till maturity in own name | Full maturity value if continued; nil if closed < 1 yr; POSA rate 1–3 yrs; NSC Rules 2019 Rule 7(4) table beyond |

*(Rule 60(10) itself notes: the scheme rules prevail and may change from time to time — scheme-wise verification is a separate topic.)*
