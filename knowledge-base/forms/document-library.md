# Topic 8 — Definitive Document Library (Official Rule Book)

**Status:** ✅ **LOCKED — ClaimSahayak Official Rule Book, Version 1.0, Topic 8 – Forms, Affidavits & Document Library** (approved 10.07.2026). Changes only via a new approved amendment entry; verification report retained at `verification-report/topic-08-document-library-verification.md`.

For each: **Purpose · When required · Who signs · Applicable schemes · Applicable scenarios · Official source.**
